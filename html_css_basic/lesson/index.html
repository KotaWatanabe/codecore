Block elements force line breaks before and after them. Examples of such elements are headings, paragraphs and list items. Block elements may only be placed inside other block elements.

Inline elements appear in the flow of text as determined by the bounds of their containing parent element. Examples of such elements are links, abbreviations, strong text emphasis and basic text emphasis.

Laws of UX (User Experience)

Laws of UX is a collection of the maxims and principles that designers can consider when building user interfaces. It was created by Jon Yablonski.

  2. Doherty Threshold
